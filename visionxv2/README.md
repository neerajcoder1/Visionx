# Vision X

> Student innovation team — building digital products & intelligent solutions.

A clean, responsive portfolio website for **Team Vision X**, showcasing team members, projects, hackathon achievements, certificates, and tech stack.

## Features

- **Team profiles** — 5 members with bios, skills, LinkedIn & GitHub links
- **Projects** — filterable gallery with live demos (TraceChain, Tariff Weaver, Food Court App, RootPilot AI, Stop Human Trafficking)
- **Achievements gallery** — grid/timeline views, category filters, photo upload, lightbox, featured carousel
- **Certificate management** — upload and display certificates with PDF/image support
- **Responsive design** — works on all screen sizes
- **Smooth animations** — scroll reveals, parallax, stat counters

## Tech Stack

- HTML5
- CSS3 (vanilla, no frameworks)
- JavaScript (vanilla)
- Google Fonts (Inter, Space Grotesk, JetBrains Mono)

## Deployment

This is a fully static site. Deploy anywhere:

```bash
# Vercel (recommended)
vercel --prod

# Or drag the folder to Netlify
```
